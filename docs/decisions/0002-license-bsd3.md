# ADR-0002: License — BSD-3-Clause

- Status: accepted (supersedes AGPL-3.0-or-later, 2026-07-10)
- Decision: D2

## Decision
The project is licensed **BSD-3-Clause**. All source carries
`// SPDX-License-Identifier: BSD-3-Clause`.

## Alternatives considered
- AGPL-3.0-or-later (original GATE 0 choice): strong copyleft; owner no longer
  wants license process overhead or CLA/dual-license open questions.
- MIT: nearly equivalent; owner preferred BSD family.

## Why
Owner directive: stop spending attention on licensing. BSD-3 is simple,
OSI-approved, and dependency-friendly. CLA/dual-commercial language is
retired; do not re-open this ADR unless the owner asks.
